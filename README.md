# SolarFit

Work out how many solar panels fit on a roof: portrait, landscape or mixed layouts from your roof dimensions. Geometry first, assumptions explicit.

**▶ Use it: https://www.steefware.com/solarfit/**

Read how it works: [How many solar panels fit on your roof?](https://www.steefware.com/blog/solar-panel-roof-layout.html)

## About

- Single self-contained `index.html` — vanilla JS, no build step, no dependencies to install
- No cookies, no ads, no accounts, no tracking; everything runs locally in your browser
- Part of [steefware.com](https://www.steefware.com/), a collection of simple free web tools

## Local development

Open `index.html` in a browser, or serve the folder:

```bash
python -m http.server
```
